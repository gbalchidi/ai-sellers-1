# Настройка email уведомлений

## Шаг 1: Создание пароля приложения Gmail

1. Перейдите на страницу настроек Google Account: https://myaccount.google.com/
2. Перейдите в раздел "Безопасность" (Security)
3. Включите двухфакторную аутентификацию (2FA), если еще не включена
4. Найдите раздел "Пароли приложений" (App Passwords)
5. Выберите "Почта" (Mail) и "Другое устройство" (Other device)
6. Введите название, например "AI Agent Landing"
7. Скопируйте сгенерированный 16-значный пароль

## Шаг 2: Настройка переменных окружения

Откройте файл `.env.local` и заполните переменные:

```env
# Gmail SMTP настройки для отправки уведомлений
GMAIL_USER=ваш-email@gmail.com
GMAIL_APP_PASSWORD=ваш-16-значный-пароль-приложения

# Получатели уведомлений о новых заявках
NOTIFICATION_EMAILS=g.balchidi@redmadrobot.com,s.shmykova@redmadrobot.com
```

**Важно:**
- `GMAIL_USER` - ваш Gmail адрес (с которого будут отправляться письма)
- `GMAIL_APP_PASSWORD` - 16-значный пароль приложения (БЕЗ пробелов)
- `NOTIFICATION_EMAILS` - список получателей через запятую

## Шаг 3: Перезапуск сервера

После заполнения `.env.local` перезапустите dev сервер:

```bash
npm run dev
```

## Тестирование

Отправьте тестовую заявку через форму на сайте. На указанные email адреса должно прийти письмо с данными заявки.

## Для продакшена

При деплое на сервер (Vercel/Railway/etc.) добавьте те же переменные окружения в настройках платформы.

## Troubleshooting

**Ошибка "Invalid login":**
- Убедитесь, что включена 2FA
- Проверьте правильность пароля приложения
- Убедитесь, что нет лишних пробелов

**Письма не приходят:**
- Проверьте папку "Спам"
- Проверьте логи сервера на наличие ошибок
- Убедитесь, что переменные окружения загружены (перезапустите сервер)
